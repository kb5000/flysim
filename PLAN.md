# FlySim — WebGL 固定翼飞行模拟器 实现规划

## 1. 目标与定位

一个浏览器内运行的固定翼飞行模拟网页游戏：

- **核心卖点：飞行手感真实**。六自由度刚体动力学 + 基于气动系数的力/力矩模型（失速、诱导阻力、副翼逆偏航、配平等都要有）。
- **画面从简**：飞机用程序生成的低多边形模型，地形用程序化高度图，无外部美术资源。
- **操作**：Xbox 手柄（Gamepad API）为主，键盘为完整备选。
- **零依赖、零构建**：原生 WebGL2 + ES Modules，任意静态服务器（`python3 -m http.server`）即可运行，不依赖 npm/CDN。

参考机型：类 Cessna 172 的单发螺旋桨教练机（参数公开、特性温和、适合调参）。

## 2. 技术选型

| 项 | 选择 | 理由 |
|---|---|---|
| 渲染 | 原生 WebGL2 | 无依赖、可控；场景简单不需要 Three.js |
| 语言 | 原生 ES Modules JS | 零构建，浏览器直接跑 |
| 物理积分 | 固定步长 120 Hz，半隐式欧拉 + 渲染插值 | 稳定、确定性好，足够精度 |
| HUD/仪表 | 独立 2D Canvas 叠加层 | 画姿态仪/速度带比 HTML 灵活 |
| 输入 | Gamepad API + KeyboardEvent | 标准 API，Xbox 手柄即插即用 |

## 3. 飞行动力学模型（核心）

### 3.1 状态量
- 位置 `p`（世界系，ENU）、速度 `v`（世界系）
- 姿态四元数 `q`（机体系→世界系）、角速度 `ω`（机体系 p/q/r）
- 机体系约定：+X 机头方向、+Y 右翼、+Z 向下（航空惯例）

### 3.2 气动力
由机体系空速求攻角 α、侧滑角 β、动压 `Q = ½ρV²`：

- **升力** `CL = CL0 + CLα·α`，带失速模型：α 超过 αstall 后 CL 平滑衰减（混合平板模型），产生真实的失速下坠与机头下沉。
- **阻力** `CD = CD0 + CL²/(π·e·AR)`（诱导阻力）+ 襟翼/起落架附加阻力。
- **侧力** `CY = CYβ·β`。

### 3.3 气动力矩（操纵性的关键）
- 俯仰：`Cm = Cm0 + Cmα·α + Cmq·q̂ + Cmδe·δe + Cmδf·flap`（静稳定 + 俯仰阻尼 + 升降舵）
- 滚转：`Cl = Clβ·β + Clp·p̂ + Clr·r̂ + Clδa·δa`（上反角效应 + 滚转阻尼 + 副翼）
- 偏航：`Cn = Cnβ·β + Cnr·r̂ + Cnp·p̂ + Cnδr·δr + Cnδa·δa`（风标稳定 + 偏航阻尼 + 方向舵 + **副翼逆偏航**）

无量纲角速率 `p̂ = pb/2V` 等，惯量用对角惯量张量（Ixx/Iyy/Izz）。

### 3.4 推进
- 螺旋桨推力随空速衰减的简化模型：`T = T_static·(1 − V/V_max)·throttle`，带一阶发动机响应（油门 spool 延迟 ~0.5s）。
- 螺旋桨滑流/扭矩的轻微偏航修正（增加真实感，可调）。

### 3.5 大气
ISA 简化：`ρ(h) = ρ0·(1 − 2.256e-5·h)^4.256`，高度越高升力/推力越小，表速/真速区分（HUD 显示 IAS）。

### 3.6 地面模型（可起飞/降落）
- 前三点式起落架：3 个接触点，弹簧-阻尼垂直支撑 + 库仑摩擦。
- 滚动摩擦/刹车摩擦区分；前轮随方向舵转向（低速地面滑行用）。
- 侧向轮胎力让滑跑保持直线、过快着陆会弹跳，接地垂速过大判定坠毁。

### 3.7 操纵增强（手感）
- 摇杆死区 0.12 + 指数曲线（expo 0.4）让小杆量更精细。
- 俯仰/副翼配平（trim），手柄十字键调节。
- 襟翼分级（0°/10°/25°），影响 CL0/CD0/Cm。

### 3.8 失效/边界
- 失速告警（α 接近 αstall 时 HUD 闪烁 + 蜂鸣，用 WebAudio 简单振荡器）。
- 超速、过载 g 表显示；坠毁后显示提示并可一键重置到跑道。

## 4. 输入映射

### Xbox 手柄（Gamepad API 标准映射）
| 控件 | 功能 |
|---|---|
| 左摇杆 X / Y | 副翼（滚转）/ 升降舵（俯仰，推杆低头） |
| LT / RT | 方向舵 左 / 右 |
| 右摇杆 Y | 油门（前推加油门，增量式） |
| A | 刹车（按住） |
| B | 襟翼收/放循环 |
| X | 重置到跑道 |
| Y | 切换视角 |
| 十字键 上/下 | 俯仰配平 |
| LB / RB | 油门 快速减/加 |
| Start | 暂停 |

手柄热插拔（`gamepadconnected` 事件），HUD 显示连接状态。

### 键盘备选
方向键=俯仰/滚转，A/D=方向舵，W/S（或 +/-）=油门，F=襟翼，B=刹车，C=视角，R=重置，P=暂停。

## 5. 渲染与世界

- **飞机模型**：代码生成的低多边形单发机（机身棱柱、机翼/平尾/垂尾平板、旋转的螺旋桨盘），操纵面（副翼/升降舵/方向舵）随输入偏转——既是视觉反馈也是调试工具。
- **地形**：基于 value-noise FBM 的高度图网格（按需分块生成，围绕飞机滑动窗口），高度着色（草地/岩石/雪线），机场周边强制压平。
- **跑道**：1000m 沥青跑道 + 中线标线 + 数字标识，出生点在跑道头。
- **天空**：全屏渐变大气穹顶 + 太阳光晕；距离雾与天色融合。
- **光照**：单方向光 + 半球环境光，无阴影（保持简单）。
- **相机**：①追逐相机（位置/朝向弹簧平滑）②机头/座舱视角（带简易座舱框）③外部环绕视角。

## 6. HUD / 仪表（2D Canvas 叠加）

- 中央：姿态指引（人工地平线 pitch ladder + 滚转刻度）、速度带（IAS, kt）、高度带（ft）、航向带。
- 角落：垂直速度、油门%、襟翼位、α、g 载荷、失速警告、手柄连接状态。
- 坠毁/暂停遮罩，简易帮助页（按 H 显示按键表）。

## 8. 实现里程碑

1. **M1 渲染骨架**：WebGL2 初始化、数学库、相机、能看到飞机模型和地面网格。
2. **M2 动力学核心**：6DOF + 气动模型 + 键盘输入，能在空中起始状态稳定飞行、做标准转弯、能进入并改出失速。
3. **M3 手柄**：Gamepad 映射、死区/expo、配平、热插拔。
4. **M4 地面与世界**：起落架物理（跑道起飞/着陆）、程序地形、跑道、天空雾。
5. **M5 HUD/相机/音频**：仪表全套、三视角、发动机音 + 失速告警音。
6. **M6 调参与验收**。

## 9. 验收标准（真实感检查单）

- [ ] 满油门从跑道滑跑，~55-60 kt 拉杆离地
- [ ] 平飞配平后松杆能保持，俯仰长周期（phugoid）振荡缓慢衰减
- [ ] 压坡度转弯需要带杆补偿掉高度，有副翼逆偏航、需踩舵协调
- [ ] 拉杆到大攻角进入失速：抖动告警→机头下沉掉高度，推杆加油门可改出
- [ ] 收油门放襟翼可控制下滑，正常拉平接地，重着陆会弹跳/坠毁
- [ ] Xbox 手柄全部映射可用，小杆量精细可控
```
